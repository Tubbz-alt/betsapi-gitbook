# Sbobet InPlay

## HTTP Request

`GET https://api.betsapi.com/v1/sbobet/inplay`

> curl "https://api.betsapi.com/v1/sbobet/inplay?sport_id=1&token=YOUR-TOKEN"

### Query Parameters

Parameter | Required? | Description
--------- | ------- | -----------
sport_id | No | R-SportID (only 1-Soccer is supported now)
page | No | R-Pager
