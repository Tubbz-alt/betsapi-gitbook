# BWin Prematch Odds

```shell
curl "https://api.betsapi.com/v1/bwin/prematch?token=YOUR_TOKEN"
```

## HTTP Request

`GET https://api.betsapi.com/v1/bwin/prematch`

### URL Parameters

Parameter | Required? | Description
--------- | ------- | -----------
day | No | format YYYYMMDD, eg: 20161201
sport_id | No | BWin sport_id
