# Team

## HTTP Request

`GET https://api.betsapi.com/v1/team`

> curl "https://api.betsapi.com/v1/team?token=YOUR_TOKEN&sport_id=1"

### URL Parameters

Parameter | Required? | Description
--------- | ------- | -----------
sport_id | Yes | R-SportID
page | No | R-Pager
