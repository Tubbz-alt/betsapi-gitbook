# Event Odds Summary

## HTTP Requst

`GET https://api.betsapi.com/v1/event/odds/summary`

> curl "https://api.betsapi.com/v1/event/odds/summary?token=YOUR_TOKEN&event_id=232751"

### URL Parameters

Parameter | Required? | Description
--------- | ------- | -----------
event_id | Yes | Event ID you get from events/*
