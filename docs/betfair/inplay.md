# Betfair InPlay

## HTTP Request

`GET https://api.betsapi.com/v1/betfair/inplay`

> curl "https://api.betsapi.com/v1/betfair/inplay?sport_id=1&token=YOUR-TOKEN"

### Query Parameters

Parameter | Required? | Description
--------- | ------- | -----------
sport_id | No | Betfair eventTypeId
page | No | R-Pager
