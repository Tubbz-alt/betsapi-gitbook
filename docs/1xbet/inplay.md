# 1xBet InPlay

## HTTP Request

`GET https://api.betsapi.com/v1/1xbet/inplay`

> curl "https://api.betsapi.com/v1/1xbet/inplay?sport_id=1&token=YOUR-TOKEN"

### Query Parameters

Parameter | Required? | Description
--------- | ------- | -----------
sport_id | No | 1xBet sport_id
page | No | R-Pager
